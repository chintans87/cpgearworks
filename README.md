# CP GEAR WORKS Premium Digital Presence

Static production website for CP GEAR WORKS, Project TITAN.

## Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Static files only
- No backend, CMS, framework, Bootstrap or jQuery

## Pages

- `index.html`
- `products.html`
- `winches.html`
- `job-work.html`
- `gallery.html`
- `contact.html`
- `404.html`

## Configuration

Replace placeholder business details in `assets/js/config.js`.

The contact form performs frontend validation only in Version 1.0. Backend delivery can be connected later at the form submission boundary in `assets/js/form-validation.js`.

## Image Strategy

Available product images and generated industrial imagery are stored under `assets/images/`. Image filenames are meaningful so future replacements can be made without changing page structure.
